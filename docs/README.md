# Docs Structure

This directory stores formal project documents.

- `analysis/`: active analysis documents for the selected main direction.
- `archive/`: archived topic analyses from earlier exploration.
- `ref/`: lecture notes, papers, and reference materials.
- `results/`: project outputs and milestone artifacts such as proposal submissions.

Document rule:
- working specifications, analysis notes, and report drafts stay under `docs/`
- script-generated experiment artifacts stay under `output/`
